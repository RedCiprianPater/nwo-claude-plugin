# Privacy Policy for NWO Robotics Claude Plugin

**Effective Date:** April 7, 2026  
**Last Updated:** April 7, 2026

## Overview

NWO Capital ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use the NWO Robotics Claude Plugin (the "Plugin").

**Plugin URL:** https://github.com/RedCiprianPater/nwo-claude-plugin  
**Website:** https://nwo.capital  
**Contact:** ciprian.pater@publicae.org

## Information We Collect

### 1. API Authentication
- **API Keys:** We store your NWO Robotics API key to authenticate requests to our services
- **Account Information:** Email address associated with your NWO Capital account

### 2. Robot Control Data
- **Robot Commands:** Natural language commands you send to control robots
- **Robot IDs:** Identifiers for robots you interact with
- **Task Parameters:** Coordinates, positions, and task specifications
- **Sensor Data:** Data from robot sensors when explicitly requested (cameras, LiDAR, IMU, etc.)

### 3. Usage Data
- **API Requests:** Logs of API calls made through the Plugin (timestamps, endpoints accessed)
- **Error Logs:** Technical error information for debugging purposes
- **Performance Metrics:** Response times and success rates

### 4. Files and Images (Optional)
- **Workspace Images:** Images you provide for VLA (Vision-Language-Action) inference
- **Configuration Files:** Task definitions, robot configurations you upload
- **Model Files:** Custom AI models you upload for robot control

## How We Use Your Information

### Primary Purposes
1. **Robot Control:** Execute your commands on connected robots
2. **VLA Inference:** Process images and natural language to generate robot actions
3. **Task Planning:** Break down complex tasks into executable steps
4. **Safety Monitoring:** Validate commands before execution to prevent accidents
5. **Billing:** Track API usage for quota management and billing

### Secondary Purposes
1. **Service Improvement:** Analyze usage patterns to improve API performance
2. **Debugging:** Investigate and fix technical issues
3. **Security:** Detect and prevent unauthorized access or abuse
4. **Support:** Assist with technical questions and troubleshooting

## Data Storage and Security

### Storage Location
- API keys are stored locally in your Claude Code configuration
- Command logs are stored on NWO Capital servers in secure data centers
- Images and files are processed in memory and not permanently stored unless explicitly saved

### Security Measures
- **Encryption:** All data transmitted between the Plugin and our API uses TLS 1.3 encryption
- **API Key Protection:** Keys are never logged or exposed in error messages
- **Access Controls:** Strict access controls on all server infrastructure
- **Regular Audits:** Security audits performed quarterly

### Data Retention
- **API Logs:** Retained for 90 days for debugging and billing
- **Error Logs:** Retained for 30 days
- **User Data:** Retained as long as your account is active
- **Deleted Data:** Permanently removed within 30 days of account deletion

## Data Sharing and Disclosure

### We Do NOT Sell Your Data
Your data is never sold to third parties for marketing or advertising purposes.

### Limited Sharing
We may share data only in the following circumstances:

1. **Service Providers:** Trusted third-party services that help us operate our infrastructure (hosting, analytics)
2. **Legal Requirements:** When required by law, court order, or government request
3. **Safety:** To prevent harm to users, robots, or property in emergency situations
4. **Business Transfers:** In connection with a merger, acquisition, or sale of assets (with notice)

### Robot Manufacturers
When you control third-party robots (Unitree, Tesla Bot, Figure AI, etc.):
- Command data is sent to the robot's control systems
- We do not share your account information with robot manufacturers
- Robot telemetry is processed by NWO Capital APIs only
